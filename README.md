### `Build:`🪜

Goal: A todolist allowing users to add items, delete items, move items to "Done" tab

Created using:
1. Bootstrap
2. EJS
3. Node.JS
4. Express
5. MongoDB Cloud
6. Heroku

### `To Run` 🏃‍♂️

1. Clone Repository
2. Open Terminal
   - $ npm i
   - $ nodemon app.js
